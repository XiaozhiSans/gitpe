# GitPe
一个半自动(?)配置git for windows的脚本

因为是写来自己~~偷懒~~用的
所以我不打算写使用手册
所以想用的话自己摸索吧awa(

目前已有功能:
- 自动配置提交者信息(也就是用户名和邮箱)
- 半自动配置ssh秘钥(需要把你的`.ssh`文件夹提前放到`main.bat`同级目录)
- 自动签名提交和标记(目前只支持ssh key ~~因为我没怎么用过gpg key(~~)
